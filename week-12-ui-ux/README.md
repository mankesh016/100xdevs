# UI/UX Foundations & Figma Design

Design is subjective, but great user experience is based on psychological principles and clean primitives.

### 🎨 Learning Objectives

- **Typography:** Understanding scale, hierarchy, and readability.
- **Color Theory:** Implementing 60-30-10 rules, contrast ratios, and emotional impact.
- **UX Laws:** Exploring Hick's Law, Fitts's Law, and the Gestalt Principles.
