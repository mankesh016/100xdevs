# Course Selling App

- **User Routes:**
  - Login
  - Signup
  - Purchase a course
  - View all available courses
  - View all purchased courses

- **Admin Routes:**
  - Login
  - Signup
  - Create a course
  - Delete a course
  - Add course content

## commands

`npm init`
`npm install express zod jsonwebtoken bcrypt mongoose dotenv`

### Define Schemas

- **UserSchema**
- **AdminSchema**
- **CourseSchema**
- **PurchaseSchema**

### .env configuration

- add mongodb connection url and jwt secrets in `.env` file
- and sample data in `.env.eample`
