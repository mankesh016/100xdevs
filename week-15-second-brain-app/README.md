# Brainly 🧠

A full-stack application for managing notes and links.

## 🛠 Tech Stack

- **Frontend:** React, TypeScript, Vite
- **Backend:** Node.js, Express, TypeScript
- **Database:** MongoDB

## 📂 Project Structure

- `Brainly-frontend`: The React client.
- `Brainly-backend`: The Express API.

## 🚀 Getting Started

### 1. Clone the repo

git clone <your-repo-url>

### 2. Setup Backend

1. Go to `Brainly-backend`.
2. Run `npm install`.
3. Create a `.env` file and add `DB_CONNECT` your mongodb url.
4. Run `npm run dev`.

### 3. Setup Frontend

1. Go to `Brainly-frontend`.
2. Run `npm install`.
3. Run `npm run dev`.
